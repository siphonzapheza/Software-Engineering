 Project Overview
**Tender Insight Hub** is a cloud-native SaaS platform designed to help South African SMEs efficiently navigate public procurement opportunities. The system simplifies complex tender documents, provides AI-powered summarization and readiness scoring, and enables team-based collaboration across different pricing tiers.

The backend will be implemented in **FastAPI**, utilizing both **SQL** (PostgreSQL or MySQL) and **NoSQL** (MongoDB or Redis) databases, simulating a modern production-grade application.
